[ text-spacing-line-height ]
Library
SuccessFailure

Line height should scale to at least 1.5 times the font size without loss of content or functionality

When line height is increased to 1.5 times the font size, text may be clipped, overlap, or overflow its container if styles like fixed heights, fixed widths, hidden overflow, or absolute positioning prevent proper reflow

Needs Fix
2 examples

Use flexible heights and widths, position elements in ways that allow text reflow, enable text wrapping, and avoid hidden overflow to make sure content remains readable when line spacing is scaled to 1.5 times the font size.

1. Review the HTML structure

2. Apply proper accessibility attributes

3. Test with screen readers
