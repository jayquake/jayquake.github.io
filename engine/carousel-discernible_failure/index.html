[ carousel-discernible ]
Library
SuccessFailure

Carousels should have a descriptive label

Carousels need a label so assistive technology announces them with a clear name, such as “Featured products carousel”. This ensures screen reader users know the widget’s purpose and can differentiate it from other carousels on the page.

Needs Fix
2 examples

Provide a clear label for the carousel using aria-labelledby to reference a visible heading or assign aria-label to the carousel container.

1. Review the HTML structure

2. Apply proper accessibility attributes

3. Test with screen readers
