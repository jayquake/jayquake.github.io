SuccessFailure

Custom checkbox controls should be tagged for assistive technology

Screen readers have built-in mechanisms to handle checkbox components. By default, screen readers don't support custom checkboxes and using them without exposing the appropriate role may prevent screen reader users from interacting as expected with the component.

Compliant
9 examples

Assign role="checkbox" to the custom checkbox element.

1. Follow proper HTML semantics

2. Ensure screen reader compatibility

3. Test with assistive technologies
