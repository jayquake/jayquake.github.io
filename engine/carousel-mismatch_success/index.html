Library
SuccessFailure

Carousels should be tagged for assistive technology

The carousel container should have a role so assistive technology treats it as a distinct section. Using role="region" with a label exposes the carousel as a named region, allowing screen reader users to recognize the purpose of the component and navigate to it efficiently.

Compliant
3 examples

Assign role="region" to the carousel container. Make sure to provide a descriptive label using aria-label or aria-labelledby.

1. Follow proper HTML semantics

2. Ensure screen reader compatibility

3. Test with assistive technologies
