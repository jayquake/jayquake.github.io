## Basis

W3 resource:
[](https://www.w3.org/WAI/standards-guidelines/act/rules/cae760/proposed/)

This rule checks that each iframe element has a non-empty <a href="https://www.w3.org/TR/accname-1.1/">accessible name</a>.
An iframes accessible name can be derived from it's title, aria-label or aria-labelledby attributes.
